# WZH-AIDev 🧠

LLM과 RAG를 활용한 AI 서비스 프로젝트들을 모아둔 곳입니다.
SK Networks Family AI Camp 13기에서의 프로젝트와 해커톤, 개인 AI 도구 등을 다루고 있습니다.

## Tech Stack

`Python` `FastAPI` `Django` `LangChain` `LangGraph` `ChromaDB` `OpenAI` `HuggingFace` `Docker` `AWS`

## Projects

| 프로젝트 | 설명 | 기술 |
|:---|:---|:---|
| **CLIKCA** *(최종 프로젝트)* | RAG 기반 업무 보조 AI 비서 시스템 | FastAPI, LangGraph, ChromaDB, Docker |
| **TravelNarae** | 문화유산 여행 코스 추천 AI 챗봇 | LangGraph, ChromaDB, 다중 API 통합 |
| **7th-kDT-HACKATHON** | 지역 문화 활성화를 위한 AI 미션 추천 플랫폼 | Django, PostgreSQL, RAG, ChromaDB |
| **SQLAI** | Gemini API 기반 MySQL 매니저 | Python, Gemini API, PyQt |
| **mySKN13_ws** | 부트캠프 학습 내용 정리 | Python, ML/DL, NLP, LangChain |
